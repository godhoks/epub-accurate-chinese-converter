# EPUB 精準漢字轉換器 · EPUB Accurate Chinese Converter

Windows 桌面工具：EPUB 簡體中文 ↔ 繁體中文互轉，可選「清除格式」。介面支援繁體中文／簡體中文／English。

## 功能

- **簡體 → 繁體 / 繁體 → 簡體**：只轉字形，**專有名詞安全**（人名不會被誤轉）
- **名詞全書同譯**：OpenCC 查表轉換天然保證；自訂詞彙表可覆蓋、全書一致
- **清除格式（選用）**：刪除出版社 CSS、內嵌字型、inline 樣式；保留章節、目錄、粗斜體、圖片
- **三語介面**：右上角 🌐 切換繁體中文／簡體中文／English，選擇會記住
- **安全**：永不覆寫原檔，輸出 `原名_traditional.epub` / `原名_simplified.epub`

## 使用

1. 執行 `AccuChineseConv.exe`（或 `python main.py`）
2. 選 EPUB 檔 → 選方向（簡 → 繁 / 繁 → 簡）
3. 視需要勾「清除格式」、選詞彙表
4. 按「開始轉換」，完成後 log 顯示輸出路徑（原檔不會被動到）

## 選項說明

### 自訂詞彙表（選填）

用來強制某個詞全書統一成你要的譯法，程式保證整本書都一致。
本工具只轉字形，若你偏好某些用語（例：把「軟件」都改成「軟體」），就用詞彙表指定。

**格式**：一個 **UTF-8 純文字檔（.txt）**，用記事本就能編輯：

```
# 井號開頭是註解，會被略過
# 一行一對：原詞=譯詞
軟件=軟體
記憶體=內存
```

**重要**：`原詞`要寫「**轉換之後**」的字形（詞彙表在字形轉換後才套用）。

**用法**：
1. 用記事本新增一個檔案，照上面格式打，另存成 `我的詞彙.txt`（編碼選 UTF-8）
2. GUI 裡「詞彙表」欄選這個檔
3. 轉換時就會自動套用

範例檔見 `glossary_example.txt`。

### 清除格式（選填）

刪除出版社的 CSS、內嵌字型、inline 樣式；保留章節、目錄、粗斜體、圖片。
適合原書排版雜亂時，讓閱讀器用自己的乾淨樣式顯示。
